# Frontend - Customer Mood Detection Interface

The frontend is a modern **React** application built with **Vite** and **Tailwind CSS**. It provides an interactive interface for recording customer interactions and viewing real-time emotion analysis.

## ✨ Features

- **Audio Recording**: High-quality audio capture using `RecordRTC`.
- **Dynamic Visuals**: Real-time feedback and emotion-driven UI elements.
- **Responsive Design**: Built with Tailwind CSS for various screen sizes.
- **Lucide Icons**: Premium iconography for an enhanced user experience.

## 🛠️ Development

1. **Install Dependencies**:
   ```bash
   npm install
   ```
2. **Run Development Server**:
   ```bash
   npm run dev
   ```
3. **Build for Production**:
   ```bash
   npm run build
   ```

## 🏗️ Technical Stack

- **React 19**: Modern component-based architecture.
- **Vite**: Ultra-fast build tool and dev server.
- **Tailwind CSS**: Utility-first CSS framework for rapid UI development.
- **Axios**: Promised-based HTTP client for API communication.
- **RecordRTC**: Robust library for recording media in the browser.

## 🔗 Connection to Backend
Ensure the backend server is running (typically at `http://127.0.0.1:8000`) for the analysis features to work.
